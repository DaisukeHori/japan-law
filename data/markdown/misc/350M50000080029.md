# 国宝、重要文化財等の管理、修理等に関する技術的指導に関する規則


*（昭和五十年文部省令第二十九号）*


<a id="第一条"></a>
**第一条**
文化財保護法（以下「法」という。）第四十七条第四項（法第八十三条において準用する場合を含む。）の規定により国宝、重要文化財又は重要有形民俗文化財の管理又は修理に関し技術的指導を求める場合には、次に掲げる事項を記載した書面をもつて行わなければならない。
国宝、重要文化財又は重要有形民俗文化財の名称及び員数
指定年月日及び指定書の記号番号又は番号
現在の所在の場所（指定書記載の所在の場所と異なる場合は、指定書記載の所在の場所を併記するものとする。）
所有者の氏名又は名称及び住所並びに法人にあつては、その代表者の氏名
管理責任者がある場合は、その氏名又は名称及び住所
管理団体がある場合は、その名称及び住所並びに代表者の氏名
技術的指導を必要とする理由
その他参考となるべき事項
前項の書面には、次に掲げる書類、図面又は写真を添えなければならない。
管理につき技術的指導を求める場合は、管理計画の概要
修理につき技術的指導を求める場合は、その設計仕様書又は計画書
現状の写真又は図面

<a id="第二条"></a>
**第二条**
法第七十条の規定により国宝、重要文化財及び登録有形文化財以外の有形文化財の管理又は修理に関し技術的指導を求める場合には、次に掲げる事項を記載した書面をもつて行わなければならない。
有形文化財の名称、種類及び員数
有形文化財が建造物であるときは、その構造、形式及び材質
有形文化財が建造物以外のものであるときは、その寸法、重量、材質その他の特徴
有形文化財に関する由来その他の説明
所在の場所
所有者の氏名又は名称及び住所並びに法人にあつては、その代表者の氏名
その他参考となるべき事項
前項の書面には、次に掲げる書類、図面又は写真を添えなければならない。
管理につき技術的指導を求める場合は、管理計画の概要
修理につき技術的指導を求める場合は、その設計仕様書又は計画書
有形文化財が建造物であるときは、その平面図
現状の写真又は図面

<a id="第三条"></a>
**第三条**
法第百十八条及び法第百二十条において準用する法第四十七条第四項の規定により特別史跡、特別名勝若しくは特別天然記念物又は史跡、名勝若しくは天然記念物（以下「史跡名勝天然記念物」と総称する。）の管理又は復旧に関し技術的指導を求める場合には、次に掲げる事項を記載した書面をもつて行わなければならない。
史跡名勝天然記念物の名称
指定年月日
所在地
所有者の氏名又は名称及び住所並びに法人にあつては、その代表者の氏名
管理責任者がある場合は、その氏名又は名称及び住所
管理団体がある場合は、その名称及び住所並びに代表者の氏名
技術的指導を必要とする理由
その他参考となるべき事項
前項の書面に添付すべき書類、図面又は写真については、第一条第二項の規定を準用する。
