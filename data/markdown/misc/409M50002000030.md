# 中小企業退職金共済法施行令の一部を改正する政令の施行に伴う経過措置に関する省令


*（平成九年労働省令第三十号）*


<a id="第一条"></a>
**第一条**
この省令は、平成十年四月一日から施行する。

<a id="第七条"></a>
**第七条**
前条の規定による改正後の経過措置省令の規定は、施行日以後に退職金の支給事由が生じた被共済者について適用し、施行日前に退職金の支給事由が生じた被共済者については、なお従前の例による。
