# 海上保安庁職員服制


*（昭和三十七年運輸省令第三十一号）*


<a id="第一条"></a>
**第一条**
海上保安庁職員（以下「職員」という。）の制服は、別表第一に定めるとおりとする。
職員の制帽は、別表第二に定めるとおりとする。

<a id="第二条"></a>
**第二条**
制服に付けるそで章、胸章又は肩章は、別表第三に定めるとおりとする。
制帽に付けるひさし章又はあごひも章は、別表第四に定めるとおりとする。

<a id="第三条"></a>
**第三条**
職員は、海上保安庁長官（以下「長官」という。）が定める場合を除き、執務中、制服及び制帽を着用しなければならない。

<a id="第四条"></a>
**第四条**
服制に関する細目は、長官が定める。
