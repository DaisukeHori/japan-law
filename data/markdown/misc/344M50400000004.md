# 警察庁の定員に関する規則

<a id="第二条"></a>
**第二条**
警察庁の各地方機関別の定員ならびに内部部局、各附属機関別および各地方機関別の警察官（皇宮警察本部にあつては、皇宮護衛官）の階級別定員は、前条に規定する定員の範囲内において、警察庁長官が定める。

<a id="第一条"></a>
**第一条**
この規則は、公布の日から施行する。
