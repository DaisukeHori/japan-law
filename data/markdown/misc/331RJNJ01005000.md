# 人事院規則一―五（特別職）


*（昭和三十一年人事院規則一―五）*


<a id="第一条"></a>
**第一条**
法第二条第三項第八号の規定に基づき、次に掲げる特別職たる機関の長の秘書官の職を特別職とする。
規則二―三（人事院事務総局等の組織）第三条第一項に規定する人事院総裁秘書官
[会計検査院法（昭和二十二年法律第七十三号）第十七条](../acts/322AC0000000073.md#第十七条)第一項に規定する秘書官のうち、会計検査院長たる検査官の秘書官
[内閣法制局設置法施行令（昭和二十七年政令第二百九十号）第七条](../cabinet_orders/327CO0000000290.md#第七条)第一項に規定する内閣法制局長官秘書官
[宮内庁法（昭和二十二年法律第七十号）第九条](../acts/322AC0000000070.md#第九条)第四項に規定する宮内庁長官秘書官

<a id="第二条"></a>
**第二条**
法第二条第三項第十号の規定に基づき、次に掲げる宮内庁の職員の職を特別職とする。
宮務主管（一人）
皇室医務主管（一人）
侍従（七人）
女官長（一人）及び女官（六人）
侍医長（一人）及び侍医（三人）
上皇侍従（六人）
上皇女官長（一人）及び上皇女官（六人）
上皇侍医長（一人）及び上皇侍医（五人）
東宮侍従長（一人）及び東宮侍従（七人）
東宮女官長（一人）及び東宮女官（六人）
東宮侍医長（一人）及び東宮侍医（三人）
皇嗣職宮務官長（一人）及び皇嗣職宮務官（十人）
皇嗣職侍医長（一人）及び皇嗣職侍医（三人）
宮務官（四人）
侍女長（四人）

<a id="第三条"></a>
**第三条**
法第二条第三項第十六号の規定に基づき、次に掲げる防衛省の職員の職を特別職から除かれる職とする。
防衛人事審議会の委員
自衛隊員倫理審査会の委員
防衛調達審議会の委員
防衛施設中央審議会の委員
防衛施設地方審議会の委員
捕虜資格認定等審査会の委員
地方協力局労務管理課の職員

<a id="第一条"></a>
**第一条**
この規則は、平成二十七年四月一日から施行する。

<a id="第十五条"></a>
**第十五条**
附則第二条から前条までに規定するもののほか、この規則の施行に関し必要な経過措置は、人事院が定める。
