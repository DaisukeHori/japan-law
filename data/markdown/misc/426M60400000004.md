# 警察大学校サイバーセキュリティ対策研究センターの内部組織に関する規則


*（平成二十六年国家公安委員会規則第四号）*


<a id="第一条"></a>
**第一条**
警察大学校サイバーセキュリティ対策研究センターに、ネットワーク解析研究室及びデバイス解析研究室を置く。

<a id="第二条"></a>
**第二条**
ネットワーク解析研究室においては、[警察法施行規則第七十七条](./329M50000002044.md#第七十七条)第二項各号に掲げる事務のうち電子情報処理組織その他の情報通信の技術及び電磁的記録（電子的方式、磁気的方式その他人の知覚によっては認識することができない方式で作られる記録であって、電子計算機による情報処理の用に供されるものをいう。）に関する研究に関する事務をつかさどる。

<a id="第三条"></a>
**第三条**
デバイス解析研究室においては、[警察法施行規則第七十七条](./329M50000002044.md#第七十七条)第二項各号に掲げる事務のうち情報通信機器及び電子機器（電子計算機及びその関連装置を除く。）に関する研究に関する事務をつかさどる。

<a id="第四条"></a>
**第四条**
ネットワーク解析研究室にネットワーク解析研究室長を、デバイス解析研究室にデバイス解析研究室長を置く。
ネットワーク解析研究室長は、命を受け、ネットワーク解析研究室の事務を掌理する。
デバイス解析研究室長は、命を受け、デバイス解析研究室の事務を掌理する。

<a id="第一条"></a>
**第一条**
この規則は、公布の日から施行する。
