# 税関職員服制


*（昭和四十四年大蔵省令第五十号）*


<a id="第一条"></a>
**第一条**
税関職員の服制は、別表に定めるところによる。

<a id="第二条"></a>
**第二条**
税関職員は、[関税法（昭和二十九年法律第六十一号）第百五条](../acts/329AC0000000061.md#第百五条)第一項（税関職員の権限）に規定する行為で次に掲げるものを行なうときは、制服を着用しないことができる。
同項第四号の二から第六号までに掲げる行為
前号に掲げるもののほか税関長が指定するもの
