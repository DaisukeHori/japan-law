# 国外における旅券手数料の額を定める省令


*（平成十八年外務省令第四号）*


<a id="第一条"></a>
**第一条**
この省令は、令和七年四月一日から施行する。

<a id="第二条"></a>
**第二条**
この省令による改正後の国外における旅券手数料の額を定める省令の規定は、この省令の施行の日以後に旅券法施行令（平成元年政令第百二十二号）第五条第一項各号、第二項各号、第三項各号及び第四項各号に掲げる処分の申請をする者に係る手数料について適用し、同日前にこれらの処分の申請をした者に係る手数料については、なお従前の例による。
