# 鉄道係員職制


*（昭和六十二年運輸省令第十三号）*


<a id="第一条"></a>
**第一条**
[鉄道営業法第十九条](../acts/133AC0000000065.md#第十九条)の規定による鉄道係員の職制については、この省令の定めるところによる。
ただし、鉄道の状況に応じ必要がある場合には、別にその職制を定めることができる。

<a id="第二条"></a>
**第二条**
鉄道に、次の運輸に関する業務を行う係員又はこれに相当する係員（以下「運輸係員」という。）を置く。
運輸長
駅長
営業係
構内係
踏切保安係
運転区長
運転士
車掌区長
車掌
運転指令
前項に規定する係員は、その職務の状況により、二以上の運輸係員の職務を兼ねることができる。

<a id="第三条"></a>
**第三条**
運輸長は、運輸に関する業務を掌理し、他の運輸係員を監督する。

<a id="第四条"></a>
**第四条**
駅長は、運輸長の命を受け、駅務を統括し、構内の秩序を保持し、その所属係員を監督する。

<a id="第五条"></a>
**第五条**
営業係は、駅長の命を受け、乗車券の発売、検査及び取集、旅客の誘導及び案内並びに荷物の取扱いの業務に従事する。

<a id="第六条"></a>
**第六条**
構内係は、駅長の命を受け、信号機及び転てつ器の取扱い並びに車両の入換えの業務に従事する。

<a id="第七条"></a>
**第七条**
踏切保安係は、駅長の命を受け、踏切の監視の業務に従事する。

<a id="第八条"></a>
**第八条**
運転区長は、運輸長の命を受け、列車の運転の業務を統括し、その所属係員を監督する。

<a id="第九条"></a>
**第九条**
運転士は、運転区長の命を受け、列車の運転の業務に従事する。

<a id="第十条"></a>
**第十条**
車掌区長は、運輸長の命を受け、列車の運転取扱い、旅客及び荷物の輸送並びに車内の秩序保持の業務を統括し、その所属係員を監督する。

<a id="第十一条"></a>
**第十一条**
車掌は、車掌区長の命を受け、列車の運転取扱い、旅客及び荷物の輸送並びに車内の秩序保持の業務に従事する。

<a id="第十二条"></a>
**第十二条**
運転指令は、運輸長の命を受け、列車の運転の整理に関する業務を行う。

<a id="第十三条"></a>
**第十三条**
鉄道に、次の工務に関する業務を行う係員又はこれに相当する係員（以下「工務係員」という。）を置く。
工務長
保線区長
保線係
営繕区長
営繕係
<a id="第二条"></a>
**第二条**第二項の規定は、工務係員について準用する。

<a id="第十四条"></a>
**第十四条**
工務長は、工務に関する業務を掌理し、他の工務係員を監督する。

<a id="第十五条"></a>
**第十五条**
保線区長は、工務長の命を受け、線路の保守の業務を統括し、その所属係員を監督する。

<a id="第十六条"></a>
**第十六条**
保線係は、保線区長の命を受け、線路の保守の業務に従事する。

<a id="第十七条"></a>
**第十七条**
営繕区長は、工務長の命を受け、線路及び電気施設以外の鉄道事業の用に供する施設の保守の業務を統括し、その所属係員を監督する。

<a id="第十八条"></a>
**第十八条**
営繕係は、営繕区長の命を受け、線路及び電気施設以外の鉄道事業の用に供する施設の保守の業務に従事する。

<a id="第十九条"></a>
**第十九条**
鉄道に、次の電気に関する業務を行う係員又はこれに相当する係員（以下「電気係員」という。）を置く。
電気長
変電区長
変電係
電路区長
電路係
信号区長
信号係
通信区長
通信係
電力指令
<a id="第二条"></a>
**第二条**第二項の規定は、電気係員について準用する。

<a id="第二十条"></a>
**第二十条**
電気長は、電気に関する業務を掌理し、他の電気係員を監督する。

<a id="第二十一条"></a>
**第二十一条**
変電区長は、電気長の命を受け、変電所等設備の保守の業務を統括し、その所属係員を監督する。

<a id="第二十二条"></a>
**第二十二条**
変電係は、変電区長の命を受け、変電所等設備の保守の業務に従事する。

<a id="第二十三条"></a>
**第二十三条**
電路区長は、電気長の命を受け、電路設備の保守の業務を統括し、その所属係員を監督する。

<a id="第二十四条"></a>
**第二十四条**
電路係は、電路区長の命を受け、電路設備の保守の業務に従事する。

<a id="第二十五条"></a>
**第二十五条**
信号区長は、電気長の命を受け、信号保安設備の保守の業務を統括し、その所属係員を監督する。

<a id="第二十六条"></a>
**第二十六条**
信号係は、信号区長の命を受け、信号保安設備の保守の業務に従事する。

<a id="第二十七条"></a>
**第二十七条**
通信区長は、電気長の命を受け、保安通信設備の保守の業務を統括し、その所属係員を監督する。

<a id="第二十八条"></a>
**第二十八条**
通信係は、通信区長の命を受け、保安通信設備の保守の業務に従事する。

<a id="第二十九条"></a>
**第二十九条**
電力指令は、電気長の命を受け、鉄道事業の運営に要する電力の供給に関する業務を行う。

<a id="第三十条"></a>
**第三十条**
鉄道に、次の車両に関する業務を行う係員又はこれに相当する係員（以下「車両係員」という。）を置く。
車両長
検車区長
検車係
工場長
修車係
<a id="第二条"></a>
**第二条**第二項の規定は、車両係員について準用する。

<a id="第三十一条"></a>
**第三十一条**
車両長は、車両に関する業務を掌理し、他の車両係員を監督する。

<a id="第三十二条"></a>
**第三十二条**
検車区長は、車両長の命を受け、車両の点検の業務を統括し、その所属係員を監督する。

<a id="第三十三条"></a>
**第三十三条**
検車係は、検車区長の命を受け、車両の点検の業務に従事する。

<a id="第三十四条"></a>
**第三十四条**
工場長は、車両長の命を受け、車両の検修の業務を統括し、その所属係員を監督する。

<a id="第三十五条"></a>
**第三十五条**
修車係は、工場長の命を受け、車両の検修の業務に従事する。
