# 困難な問題を抱える女性への支援に関する法律の施行に伴う法務省関係省令の整備及び経過措置に関する省令


*（令和六年法務省令第十号）*


<a id="第一条"></a>
**第一条**
次に掲げる省令は、廃止する。
婦人補導院処遇規則（昭和三十三年法務省令第八号）
保護具の製式（昭和三十三年法務省令第九号）
婦人補導院組織規則（平成十三年法務省令第五号）

<a id="第十二条"></a>
**第十二条**
困難な問題を抱える女性への支援に関する法律附則第十一条に規定する法務省令で定める法務省の職員は、東京西少年鑑別所庶務課の職員とする。
