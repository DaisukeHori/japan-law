# 特別高度人材の基準を定める省令


*（令和五年法務省令第二十五号）*


<a id="第一条"></a>
**第一条**
この省令は、出入国管理及び難民認定法及び日本国との平和条約に基づき日本の国籍を離脱した者等の出入国管理に関する特例法の一部を改正する法律（以下「改正法」という。）の施行の日（令和六年六月十日）から施行する。
