# 農林水産省所管の不動産登記の嘱託職員を指定する省令


*（平成十七年農林水産省令第十号）*


<a id="第一条"></a>
**第一条**
この省令は、平成二十三年九月一日から施行する。

<a id="第三条"></a>
**第三条**
この省令の施行の際現にこの省令による改正前のそれぞれの省令の規定により従前の農林水産省の機関に対してされている送付その他の行為は、この省令の施行後は、改正後のそれぞれの省令の相当規定により相当の農林水産省の機関に対してされた送付その他の行為とみなす。

<a id="第一条"></a>
**第一条**
この省令は、平成二十七年十月一日から施行する。
