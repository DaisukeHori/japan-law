# 大正十一年司法省令第四十五号（信託表示簿及日附アル印章調製方）


*（大正十一年司法省令第四十五号）*


<a id="第一条"></a>
**第一条**
公証人役場ニ備フヘキ信託表示簿及日附アル印章ハ左記雛形ニ依リ之ヲ調製スヘシ

<a id="第二条"></a>
**第二条**
信託表示簿ニハ公証人其ノ枚数ヲ表紙ノ裏面ニ記載シ職氏名ヲ署シ職印ヲ押捺シ毎葉ノ綴目ニ職印ヲ以テ契印ヲ為スコトヲ要ス
