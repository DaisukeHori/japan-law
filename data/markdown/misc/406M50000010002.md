# 検察庁法施行令第二条第一項第十一号から第十四号までの検察官の職務と密接な関連を有する職を定める省令


*（平成六年法務省令第二号）*


<a id="第一条"></a>
**第一条**
検察庁法施行令（昭和二十二年政令第三十四号。以下「令」という。）第二条第一項第十一号の職は、次に掲げる者の職とする。
上席審査専門官
審査専門官

<a id="第二条"></a>
**第二条**
令第二条第一項第十二号の職は、次に掲げる者の職とする。
特別国税査察官
統括国税査察官
国税査察官

<a id="第三条"></a>
**第三条**
令第二条第一項第十三号の職は、次に掲げる者の職とする。
統括特別調査官
主任証券取引特別調査官
証券取引特別調査官
統括証券取引特別調査官
上席証券取引特別調査官

<a id="第四条"></a>
**第四条**
令第二条第一項第十四号の職は、次に掲げる者の職とする。
統括審理官
特別審理官
犯則調査官
審理官
