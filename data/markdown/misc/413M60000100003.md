# 厚生労働省定員規則


*（平成十三年厚生労働省令第三号）*


<a id="第一条"></a>
**第一条**
厚生労働省の本省及び中央労働委員会の定員は、次の表のとおりとする。
区分
定員
備考
本省
三二、七五四人
うち、一人は、特別職の職員の定員とする。
中央労働委員会
九八人
事務局の職員の定員とする。
合計
三二、八五二人
　

<a id="第二条"></a>
**第二条**
本省の各内部部局、各施設等機関及び各地方支分部局別の定員並びに中央労働委員会の内部部局の定員は、前条に定める本省又は中央労働委員会の定員の範囲内において、厚生労働大臣が別に定める。
