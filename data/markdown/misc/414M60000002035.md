# 自動車運転代行業の業務の適正化に関する法律の施行に伴う道路交通法施行規則の規定の読替えに関する内閣府令


*（平成十四年内閣府令第三十五号）*


<a id="第一条"></a>
**第一条**
この府令は、道路交通法の一部を改正する法律（以下「改正法」という。）の施行の日（平成十九年九月十九日）から施行する。
