# 文部科学省定員規則


*（平成十三年文部科学省令第十七号）*


<a id="第一条"></a>
**第一条**
文部科学省の本省及び各外局別の定員は、次の表のとおりとする。

定員
備考
本省
一、八一三人

一〇八人
文化庁
二九一人

<a id="第二条"></a>
**第二条**
本省及び各外局の各内部部局、各施設等機関及び各特別の機関別の定員は、前条に定める本省又は各外局別の定員の範囲内において、文部科学大臣が別に定める。
