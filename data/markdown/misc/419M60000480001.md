# 特別会計に関する法律施行令第五十二条第一項第六号に規定する事務の区分を定める省令


*（平成十九年文部科学省・経済産業省令第一号）*


<a id="第一条"></a>
**第一条**
この省令は、特別会計に関する法律（平成十九年法律第二十三号）の施行の日（平成十九年四月一日）から施行する。

<a id="第二条"></a>
**第二条**
電源開発促進対策特別会計法施行令第二条第一項第三号に規定する事務の区分を定める命令（昭和五十年総理府・通商産業省令第三号）は、廃止する。
