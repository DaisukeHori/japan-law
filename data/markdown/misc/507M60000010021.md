# 刑務所、少年刑務所及び拘置所組織規則


*（令和七年法務省令第二十一号）*


<a id="第一条"></a>
**第一条**
刑務所、少年刑務所及び拘置所（以下「刑務所等」という。）の名称及び位置は、別表第一のとおりとする。

<a id="第二条"></a>
**第二条**
刑務所等に、所長を置く。
所長は、刑務所等の事務を掌理する。

<a id="第三条"></a>
**第三条**
市原青年矯正センターに、次長一人を置く。
次長は、所長を助け、市原青年矯正センターの事務を整理する。
