# 建築物エネルギー消費性能基準等を定める省令


*（平成二十八年経済産業省・国土交通省令第一号）*

**第一条**
建築物のエネルギー消費性能の向上等に関する法律（平成二十七年法律第五十三号。以下「法」という。）第二条第一項第三号の経済産業省令・国土交通省令で定める基準は、次の各号に掲げる建築物の区分に応じ、それぞれ当該各号に定める基準とする。

ただし、非住宅部分を二以上の用途に供する場合にあっては、当該非住宅部分の各用途と同一の用途の一次エネルギー消費量モデル建築物ごとに算出した設計一次エネルギー消費量を合計した数値が、当該非住宅部分の各用途と同一の用途の一次エネルギー消費量モデル建築物ごとに算出した基準一次エネルギー消費量を合計した数値を超えないこと。

<a id="第八条"></a>
**第八条**
