# 研修員手当の支給に関する規則


*（令和六年外務省令第九号）*


<a id="第一条"></a>
**第一条**
在外公館の名称及び位置並びに在外公館に勤務する外務公務員の給与に関する法律（昭和二十七年法律第九十三号）第十九条第一項に規定する外務省令で定める換算率は、支出官事務規程（昭和二十二年大蔵省令第九十四号）第十一条第二項第四号の規定により定められた外国貨幣換算率によるものとする。
