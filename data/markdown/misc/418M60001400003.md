# 特定排出者の事業活動に伴う温室効果ガスの排出量の算定に関する省令


*（平成十八年経済産業省・環境省令第三号）*


<a id="第一条"></a>
**第一条**
この省令において使用する用語は、地球温暖化対策の推進に関する法律（以下「法」という。）及び地球温暖化対策の推進に関する法律施行令（以下「令」という。）において使用する用語の例による。

**第十一条**
