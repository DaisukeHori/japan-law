# 外国相互間の貨物の移動を伴う貨物の売買、貸借又は贈与に関する取引に係る貨物が核兵器等の開発等のために用いられるおそれがある場合を定める省令


*（平成十八年経済産業省令第百一号）*


<a id="第一条"></a>
**第一条**
この省令は、平成二十一年十一月一日から施行する。

<a id="第二条"></a>
**第二条**
この省令の施行前にした行為に対する罰則の適用については、なお従前の例による。
