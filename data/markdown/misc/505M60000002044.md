# 内閣府の所管するこども家庭庁関係法令に係る国家戦略特別区域法施行規則


*（令和五年内閣府令第四十四号）*


<a id="第一条"></a>
**第一条**
この府令は、令和五年四月一日から施行する。

<a id="第二条"></a>
**第二条**

**第三条**
この府令の施行の際現にあるこの府令による改正前又は廃止前の様式（次項において「旧様式」という。）により使用されている書類は、この府令による改正後の様式によるものとみなす。
この府令の施行の際現にある旧様式による用紙については、当分の間、これを取り繕って使用することができる。
