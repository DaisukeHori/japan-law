# 海上保安大学校の名称、位置及び内部組織に関する庁令


*（昭和三十六年海上保安庁令第二号）*


<a id="第一条"></a>
**第一条**
[海上保安庁法（昭和二十三年法律第二十八号）第三十三条](../acts/323AC0000000028.md#第三十三条)の二の規定による海上保安大学校の名称、位置及び内部組織は、この庁令の定めるところによる。

<a id="第二条"></a>
**第二条**
海上保安大学校の名称及び位置は、次のとおりとする。
#### （名称）
#### （位置）
海上保安大学校
呉市

<a id="第三条"></a>
**第三条**
海上保安大学校の長は、海上保安大学校長とする。
海上保安大学校長は、海上保安大学校の校務を掌理する。

<a id="第四条"></a>
**第四条**
海上保安大学校に、副校長一名を置く。
副校長は、海上保安大学校長を助け、校務を整理し、海上保安大学校長に事故があるとき、又は海上保安大学校長が欠けたときは、臨時にその職務を行う。

<a id="第五条"></a>
**第五条**
海上保安大学校に、教授、准教授及び助教を置くほか、必要に応じ講師を置く。
教授は、専攻分野について、教育上、研究上又は実務上の特に優れた知識、能力及び実績を有する者であって、学生を教育し、研究に従事する。
准教授は、専攻分野について、教育上、研究上又は実務上の優れた知識、能力及び実績を有する者であって、学生を教育し、研究に従事する。
助教は、専攻分野について、教育上、研究上又は実務上の知識及び能力を有する者であって、学生を教育し、研究に従事する。
講師は、教授又は准教授に準ずる職務に従事する。

<a id="第五条の二"></a>
**第五条の二**
海上保安大学校に、次の講座を置く。
基礎教育
海事工学
海上警察学
海上安全学
教授、准教授、助教及び講師は、次の各号のいずれかに該当する場合を除き、前項の講座のいずれかに属するものとする。
<a id="第五条の三"></a>
**第五条の三**第十五項に規定する主任研究員又は研究員のうち、海上保安大学校長が指定する者
<a id="第五条の三"></a>
**第五条の三**第二十三項に規定するセンター教官
<a id="第十四条"></a>
**第十四条**第一項に規定する訓練教官のうち、海上保安大学校長が指定する者
第一項に掲げる講座に、それぞれ主任教授を置く。
主任教授は、命を受けて、講座に関する教務について必要な調整を行う。

<a id="第五条の三"></a>
**第五条の三**
海上保安大学校に、海上保安国際研究センター（以下この条において「センター」という。）を置く。
センターにおいては、教授、准教授、助教及び講師が行う研究及び教育のうち、次に掲げる事務をつかさどる。
国際海洋政策に関する学際的かつ総合的な研究に関すること。
海上保安庁の所掌事務に関する政策の企画及び立案に必要な高度の知識及び能力の修得並びに外国の海上保安機関の能力の向上に関する教育（以下「センター教育」という。）に関すること。
外国の海上保安機関において海上保安業務に係る実務に従事する者が当該海上保安業務を遂行するために必要な教育（以下「センター国際教育」という。）に関すること。
センターに、センター長を置く。
センター長は、センターの事務を総括する。
センター長は、海上保安庁長官が指名する者をもって充てる。
センターに、副センター長を置く。
副センター長は、センター長の職務を助ける。
副センター長は、教授のうちから、海上保安庁長官が指名する者をもって充てる。
センターに、研究調整官を置き、教授のうちから、海上保安大学校長が指名する者をもって充てる。
研究調整官は、命を受けて、センターの所掌に係る研究に関する特定事項についての企画及び立案並びに調整に関する事務をつかさどる。
センターに主任研究支援官を置き、海上保安庁長官が指名する者をもって充てる。
主任研究支援官は、研究支援官の所掌に属する事務を調整し、及びこれに関する指導を行う。
センターに研究支援官を置き、海上保安大学校長が指名する者をもって充てる。
研究支援官は、次に掲げる事務をつかさどる。
センターの行う研究の計画に関すること。
センターの行う研究に関する資料及び情報の収集、整理及び保管に関すること。
センターの行う研究の成果の普及及び活用に関すること。
センターに、主任研究員及び研究員を置き、教授、准教授、助教又は講師のうちから、海上保安大学校長が指名する者をもって充てる。
主任研究員は、研究員の研究を整理する。
研究員は、センターの行う研究に従事する。
センターに、主任教育支援官を置き、海上保安庁長官が指名する者をもって充てる。
主任教育支援官は、教育支援官の所掌に属する事務を調整し、及びこれに関する指導を行う。
センターに、教育支援官及び国際教育支援官を置き、海上保安大学校長が指名する者をもって充てる。
教育支援官は、次に掲げる事務をつかさどる。
センター教育の計画に関すること。
センター教育に関する教授等の担当する教科に関すること。
センター教育を受ける者（以下この項において「センター学生」という。）の試験及び成績に関すること。
センター学生の選考に関すること。
センター教育に関する教務の記録に関すること。
センター学生の入学、退学及び卒業に関すること。
センター教育に関する資料及び教材に関すること。
センター学生の規律、考課及び身上に関すること。
センター学生の課外活動及び学生生活に関すること。
国際教育支援官は、次に掲げる事務をつかさどる。
センター国際教育の計画に関すること。
センター国際教育に関する教授等の担当する教科に関すること。
センター国際教育を受ける者の試験及び成績に関すること。
センター国際教育に関する教務の記録に関すること。
センター国際教育に関する資料及び教材に関すること。
センター国際教育を受ける者の課外活動及び学生生活に関すること。
センター国際教育に関する施設の使用に関すること。
センターに、センター教官を置き、教授、准教授、助教又は講師のうちから、海上保安大学校長が指名する者をもって充てる。
センター教官は、センター教育に従事する。

<a id="第五条の四"></a>
**第五条の四**
海上保安大学校に、学術情報センター（以下この条において「センター」という。）を置く。
センターにおいては、次に掲げる事務をつかさどる。
学術情報システムの整備及び管理に関すること。
情報通信技術を活用した教育及び研究並びに校務の処理（以下「情報通信技術を活用した教育等」という。）の推進に関すること。
学術情報の収集、整理、保管及び提供に関すること（海上保安国際研究センターの所掌に属するものを除く。）。
センターに、センター長を置く。
センター長は、センターの事務を総括する。
センターに、副センター長を置く。
副センター長は、センター長の職務を助ける。
センター長及び副センター長は、海上保安庁長官が指名する者をもって充てる。
センターに、主任システム管理官を置き、海上保安大学校長が指名する者をもって充てる。
主任システム管理官は、システム管理官の所掌に属する事務を調整し、及びこれに関する指導を行う。
センターに、システム管理官を置き、海上保安大学校長が指名する者をもって充てる。
システム管理官は、次に掲げる事務をつかさどる。
学術情報システムの整備及び管理の実施に関すること。
学術情報システムの整備計画に関すること。
情報通信技術を活用した教育等に関する技術的事項の調査及び研究に関すること。
センターに、主任デジタル教育推進官を置き、海上保安庁長官が指名する者をもって充てる。
主任デジタル教育推進官は、デジタル教育推進官の所掌に属する事務を調整し、及びこれに関する指導を行う。
センターに、デジタル教育推進官を置き、海上保安大学校長が指名する者をもって充てる。
デジタル教育推進官は、次に掲げる事務をつかさどる。
情報通信技術を活用した教育等の推進に関する企画及び立案並びに調整に関すること。
情報通信技術を活用した教育等の推進に関する計画の策定に関すること。
センターに、主任学術情報官を置き、海上保安大学校長が指名する者をもって充てる。
主任学術情報官は、学術情報官の所掌に属する事務を調整し、及びこれに関する指導を行う。
センターに、学術情報官を置き、海上保安大学校長が指名する者をもって充てる。
学術情報官は、次に掲げる事務をつかさどる（海上保安国際研究センターの所掌に属するものを除く。）。
学術情報の収集、整理、保管及び提供の実施に関すること。
学術情報の収集に関する計画の策定に関すること。

<a id="第六条"></a>
**第六条**
海上保安大学校に、事務局、教務部、訓練部、図書館及び医務室を置く。

<a id="第七条"></a>
**第七条**
事務局に、総務課、人事厚生課及び会計課を置く。

<a id="第八条"></a>
**第八条**
総務課においては、次の事務をつかさどる。
校務（訓練部及び教務課の所掌に属するものを除く。）の総合整理に関すること。
校長の官印及び校印の管守に関すること。
文書の接受、発送及び保存に関すること。
儀式に関すること。
校内の警備及び取締りに関すること。
校内の他部課等の所掌に属さない事務に関すること。

<a id="第八条の二"></a>
**第八条の二**
人事厚生課においては、次の事務をつかさどる。
職員及び学生の任免、分限、懲戒、服務その他の人事に関すること。
職員及び学生の福利厚生に関すること。
学生の給食に関すること。

<a id="第九条"></a>
**第九条**
会計課においては、次の事務をつかさどる。
予算、決算及び会計に関すること。
国有財産の管理に関すること（訓練部の所掌に属するものを除く。）。
物品の調達及び保管並びに配分に関すること。

<a id="第十条"></a>
**第十条**
教務部に教務課を置く。

<a id="第十条の二"></a>
**第十条の二**
教務課においては、次の事務（海上保安国際研究センターの所掌に属するものを除く。）をつかさどる。
教育訓練計画に関すること（訓練課の所掌に属するものを除く。）。
教授等の担当する教科に関すること（訓練課の所掌に属するものを除く。）。
学生の試験及び成績に関すること。
入学試験に関すること。
教務の記録に関すること。
学生の入学、退学及び卒業に関すること。
教育訓練に関する資料及び教材に関すること（訓練課の所掌に属するものを除く。）。

<a id="第十一条"></a>
**第十一条**
訓練部に、学生課及び訓練課を置く。

<a id="第十二条"></a>
**第十二条**
学生課においては、次の事務（海上保安国際研究センターの所掌に属するものを除く。）をつかさどる。
学生の規律、考課及び身上に関すること。
学生の課外活動及び学生生活に関すること。
学生寮の使用に関すること。
部中の他課に属さない事務に関すること。

<a id="第十三条"></a>
**第十三条**
訓練課においては、次の事務をつかさどる。
訓練に関する教科課程及び実施計画に関すること。
訓練に関する学生の試験及び成績に関すること。
訓練に関する資料の収集及び教材の整備計画に関すること。
海上保安大学校で使用する船艇の整備及び運航管理に関すること。

<a id="第十四条"></a>
**第十四条**
訓練部に、訓練教官を置き、教授、准教授、助教又は講師のうちから充てる。
訓練教官は、学生の訓練及び生活指導に従事する。

<a id="第十五条"></a>
**第十五条**
図書館においては、海上保安大学校の図書に関する事務（学術情報センターの所掌に属するものを除く。）をつかさどる。

<a id="第十六条"></a>
**第十六条**
医務室においては、次の事務をつかさどる。
職員及び学生の保健衛生に関すること。
医療施設及び医療用品の整備計画に関すること。

<a id="第十七条"></a>
**第十七条**
この庁令に定めるもののほか、海上保安大学校の内部組織の細目は、海上保安庁長官の定めるところによる。
