# 明治八年太政官布告第百三号（裁判事務心得）


*（明治八年太政官布告第百三号）*


<a id="第三条"></a>
**第三条**
一民事ノ裁判ニ成文ノ法律ナキモノハ習慣ニ依リ習慣ナキモノハ条理ヲ推考シテ裁判スヘシ

<a id="第四条"></a>
**第四条**
一裁判官ノ裁判シタル言渡ヲ以テ将来ニ例行スル一般ノ定規トスルコトヲ得ス

<a id="第五条"></a>
**第五条**
一頒布セル布告布達ヲ除クノ外諸官省随時事ニ就テノ指令ハ将来裁判所ノ準拠スヘキ一般ノ定規トスルコトヲ得ス
