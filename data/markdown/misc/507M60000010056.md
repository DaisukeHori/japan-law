# 民法第三百八条の二の規定による子の監護費用の先取特権に係る額の算定等に関する省令


*（令和七年法務省令第五十六号）*


<a id="第一条"></a>
**第一条**
[民法第三百八条](../acts/129AC0000000089.md#第三百八条)の二に規定する法務省令で定めるところにより算定した額は、一月当たり八万円に同条に規定する定期金により扶養を受けるべき子の数を乗じて得た額とする。

<a id="第二条"></a>
**第二条**
[民法第七百六十六条](../acts/129AC0000000089.md#第七百六十六条)の三第一項に規定する法務省令で定めるところにより算定した額は、二万円に同項の規定による請求をする父母の一方が離婚の時から引き続き監護を主として行う子の数を乗じて得た額とする。
[民法第七百六十六条](../acts/129AC0000000089.md#第七百六十六条)の三第二項の規定による日割計算は、離婚の日の属する月又は同条第一項各号に掲げる日のいずれか早い日の属する月の日数を基礎としてこれを行う。
前二項の規定は、[民法第七百四十九条](../acts/129AC0000000089.md#第七百四十九条)、第七百七十一条及び第七百八十八条において同法第七百六十六条の三第一項及び第二項の規定を準用する場合について準用する。
