# 裁判官の介護休暇に関する法律


*（平成六年法律第四十五号）*


<a id="第一条"></a>
**第一条**
裁判官の介護休暇については、次条に規定するもののほか、一般職の職員の勤務時間、休暇等に関する法律（平成六年法律第三十三号）の適用を受ける職員の例に準じ、最高裁判所規則で定める。

<a id="第二条"></a>
**第二条**
裁判官は、介護休暇中は報酬を受けない。
