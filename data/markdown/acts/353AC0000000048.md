# 人質による強要行為等の処罰に関する法律


*（昭和五十三年法律第四十八号）*


<a id="第一条"></a>
**第一条**
人を逮捕し、又は監禁し、これを人質にして、第三者に対し、義務のない行為をすること又は権利を行わないことを要求した者は、六月以上十年以下の拘禁刑に処する。
第三者に対して義務のない行為をすること又は権利を行わないことを要求するための人質にする目的で、人を逮捕し、又は監禁した者も、前項と同様とする。
前項の未遂罪は、罰する。

<a id="第二条"></a>
**第二条**
二人以上共同して、かつ、凶器を示して人を逮捕し、又は監禁した者が、これを人質にして、第三者に対し、義務のない行為をすること又は権利を行わないことを要求したときは、無期又は五年以上の拘禁刑に処する。

<a id="第三条"></a>
**第三条**
航空機の強取等の処罰に関する法律（昭和四十五年法律第六十八号）第一条第一項の罪を犯した者が、当該航空機内にある者を人質にして、第三者に対し、義務のない行為をすること又は権利を行わないことを要求したときは、無期又は十年以上の拘禁刑に処する。

<a id="第四条"></a>
**第四条**
<a id="第二条"></a>
**第二条**又は前条の罪を犯した者が、人質にされている者を殺したときは、死刑又は無期拘禁刑に処する。
前項の未遂罪は、罰する。

<a id="第五条"></a>
**第五条**
<a id="第一条"></a>
**第一条**の罪は刑法（明治四十年法律第四十五号）第三条、第三条の二及び第四条の二の例に、前三条の罪は同法第二条の例に従う。
