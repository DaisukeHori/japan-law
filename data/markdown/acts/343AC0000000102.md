# 公海に関する条約の実施に伴う海底電線等の損壊行為の処罰に関する法律


*（昭和四十三年法律第百二号）*


<a id="第一条"></a>
**第一条**
公海に関する条約第二十七条に規定する海底電線（海底電信線保護万国連合条約第一条に規定する海底電信線を除く。）を損壊して電気通信を妨害した者は、五年以下の拘禁刑又は五十万円以下の罰金に処する。
過失により前項の罪を犯した者は、五十万円以下の罰金に処する。

<a id="第二条"></a>
**第二条**
公海に関する条約第二十七条に規定する海底パイプライン又は海底高圧電線を損壊して石油若しくは可燃性天然ガスの輸送又は送電を妨害した者は、五年以下の拘禁刑又は十万円以下の罰金に処する。
過失により前項の罪を犯した者は、十万円以下の罰金に処する。

<a id="第三条"></a>
**第三条**
<a id="第一条"></a>
**第一条**第一項及び前条第一項の未遂罪は、罰する。
