# 通貨及証券模造取締法


*（明治二十八年法律第二十八号）*


<a id="第一条"></a>
**第一条**
貨幣、政府発行紙幣、銀行紙幣、兌換銀行券、国債証券及地方債証券ニ紛ハシキ外観ヲ有スルモノヲ製造シ又ハ販売スルコトヲ得ス

<a id="第二条"></a>
**第二条**
前条ニ違反シタル者ハ三年以下ノ拘禁刑ニ処ス

<a id="第三条"></a>
**第三条**
<a id="第一条"></a>
**第一条**ニ掲ケタル物件ハ刑法ニ依リ没収スル場合ノ外何人ノ所有ヲ問ハス警察官ニ於テ之ヲ破毀スヘシ

<a id="第四条"></a>
**第四条**
<a id="第一条"></a>
**第一条**ニ掲ケタル物件ニハ明治九年布告第五十七号ヲ適用ス
