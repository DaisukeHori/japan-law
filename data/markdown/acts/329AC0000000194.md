# 遺産、相続及び贈与に対する租税に関する二重課税の回避及び脱税の防止のための日本国とアメリカ合衆国との間の条約の実施に伴う相続税法の特例等に関する法律


*（昭和二十九年法律第百九十四号）*


<a id="第一条"></a>
**第一条**
この法律は、遺産、相続及び贈与に対する租税に関する二重課税の回避及び脱税の防止のための日本国とアメリカ合衆国との間の条約（以下「日米相続税条約」という。）を実施するため、[相続税法](./325AC0000000073.md)（昭和二十五年法律第七十三号）の特例その他必要な事項を定めるものとする。

<a id="第二条"></a>
**第二条**
日米相続税条約第四条の規定による特定の控除は、これに相当する相続税法の規定による控除の額に同条に規定する割合を乗じて得た額に相当する額により行なうものとする。

<a id="第三条"></a>
**第三条**
政府は、日米相続税条約第一条に規定するアメリカ合衆国の租税につき、アメリカ合衆国政府から日米相続税条約第六条第二項の規定による徴収の嘱託を受けたときは、国税徴収の例により、これを徴収する。
この場合における当該租税及びその滞納処分費の徴収の順位は、それぞれ、国税及びその滞納処分費と同順位とする。

<a id="第四条"></a>
**第四条**
前二条に定めるものを除くほか、日米相続税条約の実施に関し必要な事項（この法律の規定の適用につき必要な事項を含む。）は、財務省令で定める。

<a id="第一条"></a>
**第一条**
この法律は、昭和四十年四月一日から施行する。

<a id="第一条"></a>
**第一条**
この法律は、公布の日から施行する。

<a id="第一条"></a>
**第一条**
この法律（第二条及び第三条を除く。）は、平成十三年一月六日から施行する。
ただし、次の各号に掲げる規定は、当該各号に定める日から施行する。
