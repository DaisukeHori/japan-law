# 裁判官の災害補償に関する法律


*（昭和三十五年法律第百号）*


<a id="第一条"></a>
**第一条**
この法律は、労働者災害補償保険法の一部を改正する法律（昭和四十八年法律第八十五号）の施行の日から施行する。

<a id="第一条"></a>
**第一条**
この法律は、平成八年四月一日から施行する。
