# 日本国とアメリカ合衆国との間の相互協力及び安全保障条約第六条に基づく施設及び区域並びに日本国における合衆国軍隊の地位に関する協定の実施に伴う郵便法の特例に関する法律


*（昭和二十七年法律第百二十二号）*


<a id="第一条"></a>
**第一条**
この法律は、日本国とアメリカ合衆国との間の相互協力及び安全保障条約の効力発生の日から施行する。

<a id="第一条"></a>
**第一条**
この法律は、郵政民営化法の施行の日から施行する。
