# 工場立地法施行令


*（昭和四十九年政令第二十九号）*


<a id="第一条"></a>
**第一条**
工場立地法（以下「法」という。）第六条第一項の政令で定める業種に属する工場又は事業場は、電気供給業に属する発電所で水力若しくは地熱を原動力とするもの又は太陽光を電気に変換するものとする。

<a id="第二条"></a>
**第二条**
法第六条第一項の政令で定める規模は、敷地面積については九千平方メートル、建築物の建築面積の合計については三千平方メートルとする。

<a id="第三条"></a>
**第三条**
工場適地の調査及び工場立地の動向の調査について法第十五条の三の規定により経済産業大臣が報告をさせることができる事項は、次のとおりとする。
工場又は事業場の敷地面積及び建築面積
生産数量及び生産能力
工業用水及び電力の使用の状況
燃料、原材料、外注部品及び製品の輸送の状況
従業員の雇用の状況
公害防止施設の状況
工場又は事業場の設置に関する計画又は長期の見通し
工場立地に伴う公害の防止に関する調査について法第十五条の三の規定により経済産業大臣及び環境大臣が報告をさせることができる事項は、次のとおりとする。
工場又は事業場の敷地面積
生産数量及び生産能力
生産施設、公害防止施設その他の施設の配置
燃料、原材料及び工業用水の使用の状況
汚染物質の発生の状況
汚染物質の処理その他の公害防止のための措置の内容
