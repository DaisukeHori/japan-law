# 民事保全法施行令


*（平成二年政令第二百八十四号）*


<a id="第一条"></a>
**第一条**
[民事執行法施行令（昭和五十五年政令第二百三十号）第一条](./355CO0000000230.md#第一条)の規定は、[民事保全法第四十九条](../acts/401AC0000000091.md#第四十九条)第四項において準用する民事執行法第百三十一条第三号の政令で定める額について準用する。

<a id="第二条"></a>
**第二条**
[民事執行法施行令第二条](./355CO0000000230.md#第二条)の規定は、[民事保全法第五十条](../acts/401AC0000000091.md#第五十条)第五項において準用する民事執行法第百五十二条第一項の政令で定める額について準用する。
