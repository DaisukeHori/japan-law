# 経済及び技術協力のため必要な物品等の外国政府等に対する譲与等に関する法律施行令


*（昭和四十五年政令第六十一号）*


<a id="第一条"></a>
**第一条**
経済及び技術協力のため必要な物品等の外国政府等に対する譲与等に関する法律（以下「法」という。）に規定する政令で定める財産は、船舶又は建物の従物とする。

<a id="第二条"></a>
**第二条**
法に規定する政令で定めるその他の国際機関は、日本国が加盟している国際機関及び東南アジア文部大臣機構とする。
