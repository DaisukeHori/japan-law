# 地域の自主性及び自立性を高めるための改革の推進を図るための関係法律の整備に関する法律の一部の施行に伴う厚生労働省関係政令等の整備及び経過措置に関する政令


*（平成二十三年政令第二百八十九号）*


<a id="第四条"></a>
**第四条**
地域の自主性及び自立性を高めるための改革の推進を図るための関係法律の整備に関する法律（以下「法」という。）附則第四条の政令で定める日は、令和十一年三月三十一日とする。

<a id="第五条"></a>
**第五条**
<a id="第一条"></a>
**第一条**の規定による改正後の児童福祉法施行令第四十五条の三第一項の規定により適用される児童福祉法第四十五条第一項の規定により同法第五十九条の四第一項の児童相談所設置市が条例を定める場合においては、法附則第四条中「都道府県」とあるのは、「[児童福祉法第五十九条](../acts/322AC0000000164.md#第五十九条)の四第一項の児童相談所設置市」とする。
