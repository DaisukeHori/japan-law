# 平成六年の三陸はるか沖地震による青森県八戸市の区域に係る災害についての激甚災害の指定及びこれに対し適用すべき措置の指定に関する政令


*（平成七年政令第十四号）*


<a id="第一条"></a>
**第一条**
激甚災害
適用すべき措置
平成六年の三陸はるか沖地震による災害で、青森県八戸市の区域に係るもの
法第十二条、第十三条及び第十五条に規定する措置

<a id="第二条"></a>
**第二条**
前条の激甚災害についての法第十五条第一項の政令で定める利率は、年三・六五パーセントとする。
