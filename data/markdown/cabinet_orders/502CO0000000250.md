# 令和二年五月十五日から七月三十一日までの間の豪雨による災害についての激甚災害及びこれに対し適用すべき措置の指定に関する政令


*（令和二年政令第二百五十号）*


<a id="第一条"></a>
**第一条**
激甚災害
適用すべき措置
令和二年五月十五日から七月三十一日までの間の豪雨による災害
法第三条から第六条まで、第十二条、第十六条、第十七条、第十九条、第二十条、第二十二条、第二十四条及び第二十五条に規定する措置
備考　上欄の豪雨とは、梅雨前線によるものをいう。

<a id="第二条"></a>
**第二条**

<a id="第三条"></a>
**第三条**
<a id="第一条"></a>
**第一条**の激甚災害についての令第二十五条（令第四十八条において準用する場合を含む。）の規定の適用については、令第二十五条中「第一条第一項第一号から第三号まで」とあるのは、「第一条第一項各号」とする。

<a id="第四条"></a>
**第四条**
<a id="第一条"></a>
**第一条**の激甚災害についての法第二十五条第一項ただし書の政令で定める日は、令和三年十一月十四日とする。
