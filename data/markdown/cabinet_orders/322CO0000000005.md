# 宮内庁法施行令


*（昭和二十二年政令第五号）*


<a id="第一条"></a>
**第一条**
[宮内庁法第一条](../acts/322AC0000000070.md#第一条)第二項に規定する天皇の国事に関する行為に係る事務は、[日本国憲法第七条](../constitution/321CONSTITUTION.md#第七条)第九号に規定するものに係る事務及び同条第十号に規定するものに係る事務（内閣総理大臣の定めるものを除く。）とする。

<a id="第二条"></a>
**第二条**
式部職に置かれる式部官は、儀式及び接待に関することにつき式部官長の職務を助ける。

<a id="第一条"></a>
**第一条**
この政令は、内閣法の一部を改正する法律の施行の日（平成十三年一月六日）から施行する。
