# 成年後見制度利用促進会議令


*（平成二十八年政令第二百十五号）*


<a id="第一条"></a>
**第一条**
会長は、会務を総理する。
会長に事故があるときは、会長があらかじめ指名する委員が、その職務を代理する。

<a id="第二条"></a>
**第二条**
成年後見制度利用促進会議（次条において「会議」という。）の庶務は、内閣府大臣官房企画調整課において処理する。

<a id="第三条"></a>
**第三条**
前二条に定めるもののほか、議事の手続その他会議の運営に関し必要な事項は、会長が会議に諮って定める。
