# 平成二十八年熊本地震による災害についての特定非常災害及びこれに対し適用すべき措置の指定に関する政令


*（平成二十八年政令第二百十三号）*


<a id="第一条"></a>
**第一条**
特定非常災害の被害者の権利利益の保全等を図るための特別措置に関する法律（以下「法」という。）第二条第一項の特定非常災害として平成二十八年熊本地震による災害を指定し、同年四月十四日を同項の特定非常災害発生日として定める。

<a id="第二条"></a>
**第二条**
前条の特定非常災害に対し適用すべき措置として法第三条から第七条までに規定する措置を指定する。

<a id="第三条"></a>
**第三条**
<a id="第一条"></a>
**第一条**の特定非常災害についての法第三条第一項の政令で定める日は、平成二十八年九月三十日とする。

<a id="第四条"></a>
**第四条**
<a id="第一条"></a>
**第一条**の特定非常災害についての法第四条第一項の政令で定める特定義務の不履行についての免責に係る期限は、平成二十八年七月二十九日とする。

<a id="第五条"></a>
**第五条**
<a id="第一条"></a>
**第一条**の特定非常災害についての法第五条第一項の政令で定める日は、平成三十年四月十三日とする。

<a id="第六条"></a>
**第六条**
<a id="第一条"></a>
**第一条**の特定非常災害についての法第六条の政令で定める地区は、熊本県の区域とする。
<a id="第一条"></a>
**第一条**の特定非常災害についての法第六条の政令で定める日は、平成二十八年十二月二十八日とする。

<a id="第七条"></a>
**第七条**
<a id="第一条"></a>
**第一条**の特定非常災害についての法第七条の政令で定める地区は、熊本県の区域とする。
<a id="第一条"></a>
**第一条**の特定非常災害についての法第七条の政令で定める日は、平成三十一年三月三十一日とする。
