# 農水産業協同組合の再生手続の特例等に関する法律施行令


*（平成十三年政令第三十二号）*


<a id="第一条"></a>
**第一条**
農水産業協同組合の再生手続の特例等に関する法律（以下「法」という。）第二条第二項に規定する政令で定めるものは、[農水産業協同組合貯金保険法施行令（昭和四十八年政令第二百一号）第二十四条](./348CO0000000201.md#第二十四条)に規定する貯金等とする。

<a id="第二条"></a>
**第二条**
法第四十七条に規定する政令で定めるものは、法第二十九条第一項の規定による破産手続開始の申立てとする。

<a id="第一条"></a>
**第一条**
この政令は、平成十三年四月一日から施行する。
