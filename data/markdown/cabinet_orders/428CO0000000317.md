# 電気事業法等の一部を改正する等の法律附則第三条第一項の託送供給等約款の認可の申請の期限等を定める政令


*（平成二十八年政令第三百十七号）*


<a id="第一条"></a>
**第一条**
電気事業法等の一部を改正する等の法律（次条において「改正法」という。）附則第三条第一項の政令で定める日は、平成二十八年十月三十一日とする。

<a id="第二条"></a>
**第二条**
改正法附則第九条の政令で定める日は、平成三十七年三月三十一日とする。
