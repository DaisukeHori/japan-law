# 環境影響評価法施行令


*（平成九年政令第三百四十六号）*


<a id="第一条"></a>
**第一条**
環境影響評価法（以下「法」という。）第二条第二項の政令で定める事業は、別表第一の第一欄に掲げる事業の種類ごとにそれぞれ同表の第二欄に掲げる要件に該当する一の事業とする。
ただし、当該事業が同表の一の項から五の項まで又は八の項から十三の項までの第二欄に掲げる要件のいずれかに該当し、かつ、公有水面の埋立て又は干拓（同表の七の項の第二欄に掲げる要件に該当するもの及び同表の七の項の第三欄に掲げる要件に該当することを理由として法第四条第三項第一号の措置がとられたものに限る。以下「対象公有水面埋立て等」という。）を伴うものであるときは、対象公有水面埋立て等である部分を除くものとする。

<a id="第二条"></a>
**第二条**
法第二条第二項第一号ワの政令で定める事業の種類は、宅地の造成の事業（造成後の宅地又は当該宅地の造成と併せて整備されるべき施設が不特定かつ多数の者に供給されるものに限るものとし、同号チからヲまでに掲げるものに該当するものを除く。）とする。

<a id="第三条"></a>
**第三条**
法第二条第二項第二号イの法律の規定であって政令で定めるものは、別表第一の第一欄に掲げる事業の種類（第二欄及び第三欄に掲げる事業の種類の細分を含む。）ごとにそれぞれ同表の第四欄に掲げるとおりとする。

第一項に規定する者は、同項の規定にかかわらず、経済産業大臣に書面により通知することにより、第三項の規定による判定を受けることなく法定環境影響評価等を行うことができる。
