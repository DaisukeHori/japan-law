# 石油需給適正化法施行令


*（昭和四十九年政令第十五号）*


<a id="第一条"></a>
**第一条**
石油需給適正化法（以下「法」という。）第二条第二項の政令で定める炭化水素油及び石油ガス（液化したものを含む。）は、揮発油、灯油（ジェット燃料油を含む。）、軽油、重油及びプロパン、プロピレン、ブタン又はブチレンを主成分とするガス（液化したものを含む。）とする。

<a id="第二条"></a>
**第二条**
法第七条第一項の政令で定める期間は、昭和四十九年二月から五月までの各月とする。

<a id="第三条"></a>
**第三条**
法第七条第一項第一号の政令で定める数量は、二千キロリットルとする。
