# 公証人身元保証金令


*（昭和二十四年政令第百三十九号）*


<a id="第一条"></a>
**第一条**
公証人の納むべき身元保証金の額は、次の区別に従う。
東京都の区にある区域又は大阪市に役場を設ける者
三万円
人口七万人以上の地に役場を設ける者
二万円
その他の地に役場を設ける者
一万円

<a id="第二条"></a>
**第二条**
身元保証金は、現金又は国債証券で納付しなければならない。
