# 独立行政法人海技教育機構法第十四条第一項の規定による医療法施行令の規定の技術的読替え等に関する政令


*（平成二十八年政令第五十八号）*


<a id="第一条"></a>
**第一条**
[独立行政法人海技教育機構法第十四条](../acts/411AC0000000214.md#第十四条)第一項の規定により独立行政法人海技教育機構について医療法施行令（昭和二十三年政令第三百二十六号）第一条の五の規定を適用する場合においては、同条の表の下欄中「主務大臣」とあるのは、「独立行政法人海技教育機構」とする。

<a id="第二条"></a>
**第二条**
[医療法施行令第四条](./323CO0000000326.md#第四条)の五の規定の適用については、独立行政法人海技教育機構は、国とみなす。
