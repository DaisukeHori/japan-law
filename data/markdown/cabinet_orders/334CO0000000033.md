# 内閣府設置法第四条第一項第二十四号に規定する北方地域の範囲を定める政令


*（昭和三十四年政令第三十三号）*


<a id="第一条"></a>
**第一条**
この政令は、沖縄開発庁設置法の施行の日（昭和四十七年五月十五日）から施行する。

<a id="第一条"></a>
**第一条**
この政令は、内閣法の一部を改正する法律の施行の日（平成十三年一月六日）から施行する。
