# 環境基準に係る水域及び地域の指定の事務に関する政令


*（平成五年政令第三百七十一号）*


<a id="第一条"></a>
**第一条**
[環境基本法第十六条](../acts/405AC0000000091.md#第十六条)第二項第一号の政令で定める水域は、別表に掲げる水域とする。

<a id="第二条"></a>
**第二条**
[環境基本法第四十条](../acts/405AC0000000091.md#第四十条)の二の政令で定める事務は、同法第十六条第二項の規定により都道府県又は市が処理することとされている事務のうち、交通に起因して生ずる騒音以外の騒音に係る同条第一項の基準についての同条第二項の規定による地域の指定に関する事務とする。
