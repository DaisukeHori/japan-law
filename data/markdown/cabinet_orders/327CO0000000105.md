# 鹿児島県大島郡十島村に関する鉱業法等の適用及びこれに伴う経過措置に関する政令


*（昭和二十七年政令第百五号）*


<a id="第二条"></a>
**第二条**
旧鉱業法（明治三十八年法律第四十五号）による試掘権又は採掘権であつて、その試掘鉱区又は採掘鉱区が下七島に所在するものについては、[鉱業法施行法（昭和二十五年法律第二百九十号）第一条](../acts/325AC0000000290.md#第一条)及び第二条の例による。
前項に規定する鉱業法施行法第一条の例により鉱業法による試掘権となつたものとみなされた旧鉱業法による試掘権の存続期間は、[鉱業法第十八条](../acts/325AC0000000289.md#第十八条)の規定にかかわらず、この政令の施行の日から二年とする。

<a id="第三条"></a>
**第三条**
前条に規定するものの外、鉱業法及びこれに基く命令を下七島に適用するについての経過措置は、[鉱業法施行法第四条](../acts/325AC0000000290.md#第四条)から第十七条まで、第十九条から第二十八条まで、第三十条から第四十二条まで及び第六十条に定める経過措置の例による。
