# 港則法施行令


*（昭和四十年政令第二百十九号）*


<a id="第一条"></a>
**第一条**
港則法（以下「法」という。）第二条の港及びその区域は、別表第一のとおりとする。

<a id="第二条"></a>
**第二条**
法第三条第二項に規定する特定港は、別表第二のとおりとする。

<a id="第三条"></a>
**第三条**
法第三条第三項に規定する指定港は、別表第三のとおりとする。

<a id="第一条"></a>
**第一条**
この政令は、測量法及び水路業務法の一部を改正する法律の施行の日（平成十四年四月一日）から施行する。

<a id="第一条"></a>
**第一条**
この政令は、平成十九年十二月一日から施行する。
