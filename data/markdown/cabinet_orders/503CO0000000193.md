# デジタル社会推進会議令


*（令和三年政令第百九十三号）*


<a id="第一条"></a>
**第一条**
議長は、会務を総理する。

<a id="第二条"></a>
**第二条**
副議長は、議長を助け、議長に事故があるときは、その職務を代理する。

<a id="第三条"></a>
**第三条**
デジタル社会推進会議の庶務は、デジタル庁に置かれる統括官が処理する。

<a id="第四条"></a>
**第四条**
前三条に定めるもののほか、議事の手続その他デジタル社会推進会議の運営に関し必要な事項は、議長がデジタル社会推進会議に諮って定める。
