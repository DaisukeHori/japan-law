# 有限責任事業組合契約に関する法律施行令


*（平成十七年政令第二百六十九号）*


<a id="第一条"></a>
**第一条**
有限責任事業組合契約に関する法律（以下「法」という。）第七条第一項第一号に規定するその性質上組合員の責任の限度を出資の価額とすることが適当でない業務として政令で定めるものは、次に掲げるものとする。
[公認会計士法（昭和二十三年法律第百三号）第二条](../acts/323AC0000000103.md#第二条)第一項に規定する業務
[弁護士法（昭和二十四年法律第二百五号）第七十二条](../acts/324AC1000000205.md#第七十二条)本文の規定により弁護士又は弁護士法人でない者が行うことができない業務
[司法書士法（昭和二十五年法律第百九十七号）第三条](../acts/325AC1000000197.md#第三条)第一項に規定する業務
[土地家屋調査士法（昭和二十五年法律第二百二十八号）第三条](../acts/325AC1000000228.md#第三条)第一項に規定する業務
[行政書士法（昭和二十六年法律第四号）第十九条](../acts/326AC1000000004.md#第十九条)第一項本文の規定により行政書士又は行政書士法人でない者が行うことができない業務
[海事代理士法（昭和二十六年法律第三十二号）第一条](../acts/326AC0000000032.md#第一条)に規定する業務
[税理士法（昭和二十六年法律第二百三十七号）第二条](../acts/326AC1000000237.md#第二条)第一項及び第二条の二第一項に規定する業務
[社会保険労務士法（昭和四十三年法律第八十九号）第二条](../acts/343AC1000000089.md#第二条)第一項第一号から第二号までに掲げる業務
[弁理士法（平成十二年法律第四十九号）第四条](../acts/412AC0000000049.md#第四条)第二項、第五条第一項、第六条及び第六条の二第一項に規定する業務並びに同法第七十五条の規定により弁理士又は弁理士法人でない者が行うことができない業務

<a id="第二条"></a>
**第二条**
法第七条第一項第二号に規定する組合の債権者に不当な損害を与えるおそれがある業務として政令で定めるものは、次に掲げるものとする。
[当せん金付証票法（昭和二十三年法律第百四十四号）第二条](../acts/323AC0000000144.md#第二条)第一項に規定する当せん金付証票の購入
[競馬法（昭和二十三年法律第百五十八号）第六条](../acts/323AC0000000158.md#第六条)第一項及び第二項（同法第二十二条において準用する場合を含む。）の勝馬投票券の購入
[自転車競技法（昭和二十三年法律第二百九号）第八条](../acts/323AC1000000209.md#第八条)の車券の購入
[小型自動車競走法（昭和二十五年法律第二百八号）第十二条](../acts/325AC1000000208.md#第十二条)の勝車投票券の購入
[モーターボート競走法（昭和二十六年法律第二百四十二号）第十条](../acts/326AC1000000242.md#第十条)第一項及び第二項の舟券の購入
スポーツ振興投票の実施等に関する法律（平成十年法律第六十三号）第八条第一項及び第二項のスポーツ振興投票券の購入

<a id="第一条"></a>
**第一条**
この政令は、法の施行の日（平成十七年八月一日）から施行する。

<a id="第一条"></a>
**第一条**
この政令は、特許法等の一部を改正する法律（以下「改正法」という。）の施行の日（令和四年四月一日）から施行する。
