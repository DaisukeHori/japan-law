📋 法案 = GitHub Issues

法律の改正提案をIssueとして管理

💡 法律 = ソースコード

このプロジェクトでは、法律をソースコードのように Git で管理しています。 法案（Bill）は、ソフトウェア開発における「Issue」や「Pull Request」に相当します。

🔄 ソフトウェア開発

  • Issue: バグ報告・機能要望
  • Pull Request: コード変更の提案
  • Merge: 変更の承認・反映
  • git diff: 変更差分の確認
  • git log: 変更履歴の追跡

⚖️ 立法プロセス

  • Issue: 法案提出・審議開始
  • Pull Request: 法案の内容（新旧対照表）
  • Merge: 法案成立・公布
  • git diff: 改正前後の差分
  • git log: 改正履歴

📝 法案を Issue として管理

法案の提案・議論は GitHub Issues で行います。 ラベルを使って状態（審議中・成立・廃案）を管理できます。

Issues を見る ➕ 新しい法案を提案 🔀 Pull Requests
